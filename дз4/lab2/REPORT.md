# Отчет по лабораторной работе №2

## Что реализовано

1. Отображение данных запроса:
- страница параметров URL: `/request/url`;
- страница заголовков запроса: `/request/headers`;
- страница cookie: `/request/cookies`;
- форма авторизации с выводом отправленных данных: `/request/form`.

2. Форма проверки номера телефона:
- страница `/phone`, отправка методом `POST`;
- проверка допустимых символов: цифры, пробелы, `()`, `-`, `.`, `+`;
- проверка количества цифр:
  - 11 цифр, если номер начинается с `+7` или `8`;
  - 10 цифр в остальных случаях;
- ошибки валидации отображаются через Bootstrap-классы `is-invalid` и `invalid-feedback`;
- успешный ввод преобразуется в формат `8-***-***-**-**`.

## Запуск

```bash
python -m venv ve
source ve/bin/activate
pip install -r requirements.txt
python app.py
```

Приложение откроется на `http://127.0.0.1:5000`.
